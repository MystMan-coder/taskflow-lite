# TaskFlow Lite 🚀

A lightweight task management system with async notifications, event-driven architecture, and cloud deployment.

## ✨ Features

* User authentication (JWT)
* Project & task management
* Task status tracking
* File uploads (AWS S3)
* Dashboard analytics (Redis caching)
* Event-driven architecture (Kafka)
* Async notifications (RabbitMQ)

## 🏗️ Tech Stack

### Backend

* Spring Boot (Java)
* PostgreSQL (AWS RDS)
* Redis
* Kafka
* RabbitMQ

### Frontend

* React + TypeScript
* Redux + Context API
* Tailwind CSS

### DevOps

* Docker
* AWS EC2
* AWS S3
* AWS Lambda

## 🚀 Getting Started

```bash
docker-compose up --build
```
